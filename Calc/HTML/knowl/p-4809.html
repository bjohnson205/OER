The rule for \(S_{2n}\) giving by Equation (5.15) is usually known as Simpson's Rule. 9  Note that we use “\(S_{2n}\)” rather that “\(S_n\)” since the \(n\) points the Midpoint Rule uses are different from the \(n\) points the Trapezoid Rule uses, and thus Simpson's Rule is using \(2n\) points at which to evaluate the function. We build upon the results in Table 5.29 to see the approximations generated by Simpson's Rule. In particular, in Table 5.30, we include all of the results in Table 5.29, but include additional results for \(S_8 = \frac{2M_4 + T_4}{3}\) and \(S_{16} = \frac{2M_8 + T_8}{3}\text{.}\)

The graph below is a plot of \(\frac{dy}{dt}\) versus \(y\text{.}\) We see that when \(y \lt 4\text{,}\) \(\frac{dy}{dt} \gt 0\) and hence \(y\) is an increasing function of \(t\text{.}\) We also see that when \(y \gt 4\text{,}\) \(\frac{dy}{dt} \lt 0\) and hence \(y\) is a decreasing function of \(t\text{.}\)

  2. Next, by sampling points \((t,y)\) in the plane and plotting short lines whose slope is \(\left. \frac{dy}{dt}\right|_{(t,y)}\text{,}\) we create the following slope field for the differential equation \(\frac{dy}{dt} = -\frac{1}{2}(y - 4)\text{.}\)

  3. Finally, by choosing starting points and following the direction of the slope field, we generate plots of solutions that satisfy \(y(0) = 0\text{,}\) \(y(0) = 2\text{,}\) \(y(0) = 4\text{,}\) and \(y(0) = 6\text{,}\) as shown in the following figure.

  4. For \(y(t) = 4 + 2e^{-t/2}\text{,}\) we see that

    \begin{equation*} \frac{dy}{dt} = 2 \left( -\frac{1}{2} e^{-t/2} \right) = -e^{-t/2} \end{equation*}

    and

    \begin{equation*} -\frac{1}{2}( y - 4 ) = -\frac{1}{2} \left( 4 + 2e^{-t/2} \right) = -e^{-t/2} \end{equation*}

    In addition, \(y(0) = 4 + 2e^0 = 6\text{.}\) So \(y(t) = 4 + 2e^{-t/2}\) is a solution of the differential equation \(\frac{dy}{dt} = -\frac{1}{2}(y - 4)\) with \(y(0) = 6\text{.}\) The graph of this function is identical to the top graph in part (c).

  5. The solution where \(y(0) = 4\) is a constant function.

In this case the differential equation becomes

\begin{equation*} \frac{dP}{dt} = g(P) = P(6-P)-h\text{.} \end{equation*}

We will have equilibrium solutions when \(P^2 - 6P + h = 0\) or when

\begin{equation*} P = \frac{6+\sqrt{36-4h}}{2} \ \ \ \text{ and } \ \ \ P = \frac{6-\sqrt{36-4h}}{2}\text{.} \end{equation*}

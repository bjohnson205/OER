The solution for the general logistic initial value problem is

\begin{equation*} P(t) = \frac{N}{\left( \frac{N - P_0}{P_0} \right) e^{-kNt} + 1}\text{.} \end{equation*}

For this solution, we see that

\begin{align*} P(0) \amp = \frac{N}{\left( \frac{N - P_0}{P_0} \right) + 1}\\ \amp = \frac{N}{\frac{N - P_0 + P_0}{P_0}}\\ \amp = \frac{N P_0}{N}\\ \amp = P_0 \end{align*}

To determine \(\lim_{t \to \infty} P(t)\text{,}\) we first note that since \(-kN \lt 0\text{,}\) \(\lim_{t \to \infty} e^{-kNt} = 0\text{.}\) Hence,

\begin{equation*} \lim_{t \to \infty}\left[ \left( \frac{N - P_0}{P_0} \right)e^{-kNt} + 1 \right] = 1\text{,} \end{equation*}

and therefore,

\begin{equation*} \lim_{t \to \infty} P(t) = \lim_{t\to \infty} \frac{N}{\left( \frac{N - P_0}{P_0} \right) e^{-kNt} + 1} = \frac{N}{1} = N\text{.} \end{equation*}

Item7.4.9.d

Changing the initial condition to \(P(0) = 6\text{,}\) we can use all of our preceding work in (c) up to where we established \(P = 3e^{Ke^{-t}}\text{.}\) Applying this initial condition, \(6 = 3e^K\text{,}\) so \(K = \ln \left(2 \right)\text{,}\) and thus

\begin{equation*} P(t) = 3e^{\ln \left( 2 \right) e^{-t}}\text{.} \end{equation*}

This function's plot is shown in red in the image above in (a) and we see (both graphically and algebraically) that \(P(t) \to 3\) as \(t \to \infty\) since \(e^{-t} \to 0\text{.}\)

We first note that \(15^{\circ}\) is half of \(30^{\circ}\text{.}\) Therefore,

\begin{equation*} \cos(15^{\circ})=\cos\left(\frac{30^{\circ}}{2}\right) = \pm \sqrt{\frac{1+\cos(30^{\circ})}{2}}. \end{equation*}

We can evaluate the cosine. Since 15 degrees is in the first quadrant, we need the positive result.

\begin{equation*} \cos(15^{\circ}) = \sqrt{\frac{1+\frac{\sqrt{3}}{2}}{2}} = \sqrt{\frac{1}{2}+\frac{\sqrt{3}}{4}} \end{equation*}

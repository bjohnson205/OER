Solution33.1

Substituting the known value for sine into the Pythagorean identity, we have

\begin{align*} \cos^2(\theta) + \left(\frac{3}{7}\right)^2 \amp= 1 \amp\amp \text{Simplifying}\\ \\ \cos^2(\theta) + \frac{9}{49} \amp= 1 \amp\amp \text{Subtracting } \frac{9}{49} \text{ from both sides}\\ \\ \cos^2(\theta) \amp= \frac{40}{49} \amp\amp \text{Taking the square root of both sides }\\ \\ \cos(\theta) \amp= \pm \sqrt{\frac{40}{49}} \amp\amp \text{Simplifying} \\ \\ \amp= \pm \frac{\sqrt{40}}{7} \\ \\ \amp= \pm \frac{2\sqrt{10}}{7} \end{align*}

Since \(\theta\) is in the second quadrant, we know the \(x\) value of the corresponding point on the unit circle is negative. Therefore, the cosine value should also be negative. Using this additional information, we can conclude that

\begin{equation*} \cos(\theta) = -\frac{2\sqrt{10}}{7} \end{equation*}

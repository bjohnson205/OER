Since there are no common angles on the unit circle for which \(\tan(\theta)=3\text{,}\) we must apply the inverse tangent function and solve for \(\theta\) approximately. We get that

\begin{equation*} \theta = \tan^{-1}(3) \end{equation*}

Using our calculator to approximate this angle, we get

\begin{equation*} \theta \approx 1.249 \text{ radians} \end{equation*}

To check this solution, we can evaluate \(\tan(1.249)\) to see if we get a value close to 3. Using our calculator we get

\begin{equation*} \tan(1.249) = 2.9995 \end{equation*}

Since this value is close to 3, we can be confident that the angle we have found satisfies the equation \(\tan(\theta)=3\text{.}\)

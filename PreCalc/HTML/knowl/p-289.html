The notation \(f(x)\) is read as, “\(f\) of \(x\)” and should not be confused with multiplication. Algebra frequently involves functions, and so the notation becomes useful when performing common tasks. Here \(f\) is the function name, and \(f(x)\) denotes the value in the range associated with the value \(x\) in the domain. Functions are often named with different letters; some common names for functions are \(f, g, h, C\text{,}\) and \(R\text{.}\) We have determined that the set of solutions to \(y=|x|-2\) is a function; therefore, using function notation we can write \(\alert{f(x)}=|x|-2\) instead of \(y=|x|-2\text{.}\)

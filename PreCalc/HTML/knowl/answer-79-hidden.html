Consider the function \(f(x)=2x^2+3x\text{.}\) There are two solutions to \(f(x)=-1\text{:}\)

\begin{align*} 2x^2+3x \amp = -1 \\ 2x^2+3x+1 \amp = 0 \\ (x+1)(2x+1) \amp = 0. \end{align*}

Therefore, we get the two solutions

\begin{equation*} x=-1 \text{ and } x=-\frac{1}{2}. \end{equation*}

Following these steps with \(\sin(\theta)\) instead of \(x\text{,}\) we know that either

\begin{equation*} \sin(\theta)=-1 \text{ or } \sin(\theta)=-\frac{1}{2}. \end{equation*}

\(\sin(\theta)=-1\) is solved by

\begin{equation*} \theta = \frac{3\pi}{2} + 2\pi k \hspace{.25in} \text{ where } k \text{ is any integer}. \end{equation*}

Solving the equation \(\sin(\theta)=-\frac{1}{2}\text{,}\) we get two solution sets:

\begin{align*} \theta \amp = \frac{7\pi}{6} + 2\pi k \hspace{.25in} \text{ where } k \text{ is any integer} \\ \\ \theta \amp = \frac{11\pi}{6} + 2\pi k \hspace{.25in} \text{ where } k \text{ is any integer}. \end{align*}

Finally, we need to determine which solutions lie in the interval \([0,2\pi]\text{.}\) This happens for \(k=0\) with all of the solutions sets listed above, so the solutions to the original equation \(2\sin^2(\theta)+3\sin(\theta) = -1\) in the interval \([0,2\pi]\) are

\begin{equation*} \theta = \frac{3\pi}{2}, \hspace{.25in} \frac{7\pi}{6}, \hspace{.25in} \text{ and } \hspace{.25in} \frac{11\pi}{6}. \end{equation*}

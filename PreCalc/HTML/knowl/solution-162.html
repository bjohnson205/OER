Solution173.1

First we set up a function representing the average cost.

\begin{equation*} A(x)=\frac{C(x)}{x}=\frac{0.01x^2−3x+1200}{x} \end{equation*}

Next, we calculate \(A(100), A(200)\text{,}\) and \(A(300)\text{.}\)

\begin{equation*} \begin{aligned} A(100)\amp =\frac{0.01(\alert{100})^2−3(\alert{100})+1200}{(\alert{100})}=\frac{100-300+1200}{100}=\frac{1000}{100}=10\\ A(200)\amp =\frac{0.01(\alert{200})^2−3(\alert{200})+1200}{(\alert{200})}=\frac{400-600+1200}{200}=\frac{1000}{200}=5\\ A(300)\amp =\frac{0.01(\alert{300})^2−3(\alert{300})+1200}{(\alert{300})}=\frac{900-900+1200}{300}=\frac{1200}{300}=4\\ \end{aligned} \end{equation*}

The average cost of producing \(100\) sweaters per day is \($10.00\) per sweater. If \(200\) sweaters are produced, the average cost per sweater is \($5.00\text{.}\) If \(300\) are produced, the average cost per sweater is \($4.00\text{.}\)

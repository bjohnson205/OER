Given the algebraic equation of a line, we can graph it in a number of ways. In this section, we will be given a geometric description of a line and find the algebraic equation. Finding the equation of a line can be accomplished in a number of ways. The following example makes use of slope-intercept form, \(y=mx+b\text{,}\) or using function notation, \(f(x)=mx+b\text{.}\) If we can determine the slope, \(m\text{,}\) and the \(y\)-intercept, \((0,b)\text{,}\) we can then construct the equation.

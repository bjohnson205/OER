Real-world applications are often modeled using more than one variable and more than one equation. A system of equations consists of a set of two or more equations with the same variables. In this section, we will study linear systems consisting of two linear equations each with two variables. The system

\begin{equation*} \begin{cases} 2x-3y\amp =0\\ -4x+2y\amp =-8\\ \hline \end{cases} \end{equation*}

is one such system.

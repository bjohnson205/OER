In Table 176 you can see that as the \(x\)-values decrease toward negative infinity, the corresponding \(y\)-values decrease toward zero. As a result, the graph of \(f\) decreases toward the \(x\)-axis as we move to the left. Thus, the \(x\)-axis, \(y=0 \text{,}\) is a horizontal asymptote for exponential functions no matter what value \(b\text{,}\) the growth factor, happens to be. We can see this illustrated in Figure 178, since the \(y\)-values of each exponential get closer to zero.

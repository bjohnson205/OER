Here we separate the domain (\(x\)-values), and the range (\(y\)-values), and depict the correspondence between the values with arrows.

The relation is a function because each \(x\)-value corresponds to exactly one \(y\)-value. The domain is \(\{−1, 0, 2, 3, 4\}\) and the range is \(\{−2, 3, 4, 7\}\text{.}\)
